Why learn Java?

Java is what you could call a "gateway language". Lots of elements from Java can apply to many other languages. Many concepts and rules in Java. It's also not very hard to learn. though some topics may be challenging. This site was made for beginners. This website will have interactive pages and examples that should help you understand the basic concepts of Java with relative ease.

Where can I write Java?

When I learned I used a compiler called OnlineGDB. It's a good compiler that can tell you any errors you made and gives you some form of an explanation on what you did wrong and where you did something wrong. This is especially good for Java because at the end of almost every line you need to add a semicolon. One of these things.

;

Forgetting to do so will result in an error. Just make sure that you create an account so you can save projects and select Java as your coding language on the top right drop-down menu. Most compilers should be good but this is what I prefer even though it is slightly gltchy and oudated.

Rules of thumb:

  • As mentioned above, add a semicolon at the end of every line unless told otherwise.
  • Close every pair parenthesis and brackets.
  • Spell everything right: If you spell anything wrong it will result in an error. This holds true in most coding languages.
